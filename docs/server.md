# Server details

The hostname, os and uptime can be found at **/api/server/hostname**, 
**/api/server/os** and **/api/server/uptime** respectively.

```javascript
{
    "hostname": "raspberrypi"
}
```

```javascript
{
    "os_name": "Linux-4.4.26-v7+-armv7l-with-debian-8.0"
}
```

```javascript
{
    "uptime": "11 days, 16:43:08"
}
```
